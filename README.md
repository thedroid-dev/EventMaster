# 🎉 EventMaster

EventMaster es un plugin inspirado en ConditionalEvents para demostrar sistemas de eventos en servidores Minecraft.

---

## 🚀 Características
- Eventos automáticos configurables  
- Soporte para recompensas y anuncios  
- Integración con economía y rangos  
- Configuración en `config.yml`  

---

## 📦 Instalación
1. Coloca `EventMaster.jar` en la carpeta `plugins/`.  
2. Reinicia el servidor.  
3. Configura eventos y recompensas en `config.yml`.  

---

## 📜 Comandos
| Comando   | Descripción             | Permiso            |
|-----------|-------------------------|--------------------|
| `/event`  | Administra eventos      | `eventmaster.event`|
